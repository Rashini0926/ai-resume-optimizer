# AI Resume Optimizer

An AI-powered tool that analyzes a resume against a job description, gives
an ATS-compatibility score with improvement suggestions, and helps track
job/internship applications in one place.

Built as a full-stack portfolio project demonstrating frontend, backend,
database, testing, and DevOps skills end-to-end.

## Tech Stack

| Layer      | Technology                     |
|------------|---------------------------------|
| Frontend   | React + TypeScript (Vite)      |
| Backend    | Node.js + Express + TypeScript |
| Database   | MongoDB                        |
| AI         | Claude / OpenAI API            |
| Auth       | JWT-based authentication       |
| Testing    | Jest, Postman/Newman, Cypress  |
| CI/CD      | GitHub Actions                 |
| Deployment | Vercel + Railway               |
